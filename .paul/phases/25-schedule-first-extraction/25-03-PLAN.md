<task type="auto">
  <name>Task 1: Extract Remaining MEP Schedule Sheets (20 non-Architectural)</name>
  <files>pipeline/extraction_harness.py</files>
  <action>
    Using the harness, process the 20 non-Architectural remaining sheets in batches:

    **Batch order (by discipline priority):**
    - Refrigeration: 1 sheet (R6002 or similar)
    - Utility: 3 sheets
    - Mechanical: 3 sheets
    - Electrical: 10 sheets
    - Structural: 3 sheets

    For each batch:
    1. Call h.next_sheet() to get the next pending sheet
    2. Use a pdfplumber agent to read the PDF and extract schedule tables
    3. Normalize extracted data into harness-compatible format
    4. Call h.record_result(sheet_id, entries)
    5. After each batch of 5, save checkpoint

    **Parallel agents**: Launch up to 5 pdfplumber agents concurrently for each batch.

    **Empty sheets**: Some schedule drawings may have calculation tables or notes instead of equipment schedules. Record these with an empty entries list — do not fabricate equipment.

    **Error handling**: If a PDF fails to extract, call h.record_error(sheet_id, error_msg). The harness retries once, then skips.
  </action>
  <verify>
    get_schedule_summary() shows 20+ additional sheets extracted beyond the initial 5.
    Discipline breakdown includes Refrigeration, Utility, Mechanical, Electrical, Structural entries.
  </verify>
  <done>AC-1 partially satisfied (MEP sheets done). AC-2 satisfied: MEP disciplines extracted with cross-references.</done>

<task type="auto">
  <name>Task 2: Extract Architectural Schedule Sheets (34 sheets)</name>
  <files>pipeline/extraction_harness.py</files>
  <action>
    Process the 34 Architectural schedule sheets. These are primarily:
    - **Door schedules** (A6001-A6004): door tags, sizes, hardware, fire ratings, frame types
    - **Window schedules**: window tags, types, sizes, glazing specs
    - **Finish schedules**: room numbers, floor/wall/ceiling finishes
    - **Other**: equipment pads, roof hatches, misc architectural schedules

    For each sheet:
    1. Read PDF with pdfplumber agent
    2. Extract schedule table data
    3. Map to equipment entries:
       - Door tags → equipment_type "Door"
       - Window tags → equipment_type "Window"
       - Finish schedules → equipment_type "Room Finish" with room number as tag
       - If no extractable schedule tables, record empty result
    4. Store via harness

    **Key**: Architectural schedules often have many rows (100+ doors). Extract ALL rows — the schedule_extractions table can handle volume.

    **Parallel agents**: Launch up to 5 pdfplumber agents concurrently per batch.
  </action>
  <verify>

    Architectural discipline has entries from door/window/finish schedules.
  </verify>
