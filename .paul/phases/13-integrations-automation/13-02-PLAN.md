<objective>
## Goal
Add cross-module credential links (welding certifications in the employee credential portfolio) and bulk operations (mass renewal processing + batch CE credit entry) to the licenses module.

## Purpose
Completes the integration story: employees see ALL their credentials (state licenses + welding qualifications) in one place, and admins can process renewals and CE credits in bulk instead of one-by-one.

## Output
- Credential portfolio shows welding certifications alongside state licenses
- Bulk renewal endpoint + UI for processing multiple license renewals at once
- Batch CE credit entry endpoint + UI for entering CE credits for multiple employees
</objective>

    - Add get_employee_welding_credentials(conn, employee_id) function
    - Query: SELECT wwr.employee_number, wwr.welder_stamp, wwr.first_name, wwr.last_name,
             wpq.id as wpq_id, wpq.welding_process, wpq.position, wpq.base_material_group,
             wpq.qualification_date, wpq.expiration_date as wpq_expiration,
             wwr.continuity_status

<task type="auto">
  <name>Task 3: Bulk renewal and batch CE credit entry</name>
  <files>licenses/db.py, api/licenses.py, frontend/templates/licenses/licenses.html</files>
  <action>
    In licenses/db.py:
    - Add bulk_renew_licenses(conn, license_ids, new_expiration, notes, created_by):
      For each license_id: update expiration_date, insert license_event (type='renewed'),
      update status to 'active' if was 'expired'. Return count of renewed.
    - Add batch_create_ce_credits(conn, employee_id, credits_data, created_by):
      credits_data is list of {course_name, hours, completion_date, provider, license_ids}
      Insert into ce_credits for each entry. Return count of credits created.

    In api/licenses.py:
    - POST /api/bulk-renew: accepts {license_ids: [], new_expiration: str, notes: str}
      Calls bulk_renew_licenses. Returns {renewed: N}.
    - POST /api/batch-ce: accepts {employee_id: str, credits: [{course_name, hours, completion_date, provider}]}
      Calls batch_create_ce_credits. Returns {created: N}.
