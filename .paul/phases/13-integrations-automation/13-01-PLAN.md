<objective>
## Goal
Add license compliance widgets to the home dashboard and create an iCal calendar feed for renewal/CE deadlines — making license status visible at a glance and exportable to external calendars.

## Purpose
Licenses module currently contributes nothing to the home dashboard despite being the most active module. Users must navigate to `/licenses/` to see expiring licenses or CE alerts. An iCal feed enables Outlook/Google Calendar integration without building a custom scheduler.

## Output
- Home dashboard: expiring licenses count + CE alerts stat cards (module-gated)
- New endpoint: `GET /licenses/calendar.ics` returning iCalendar feed
- "Subscribe to Calendar" link on licenses main page
</objective>

<task type="auto">
  <name>Task 2: Create iCal calendar feed endpoint</name>
  <files>api/licenses.py, licenses/db.py</files>
  <action>
    In licenses/db.py, add function `get_calendar_events(months_ahead=12)`:
    - Query v_expiring_licenses WHERE expiration_date BETWEEN date('now') AND date('now', '+N months')
    - Also query ce_requirements joined with state_licenses for CE period end dates
    - Return list of dicts with: uid, summary, description, dtstart, location (state)

    In api/licenses.py, add route:
    - GET /licenses/calendar.ics
    - Protected by @module_required("licenses")
    - Call get_calendar_events()
    - Build iCalendar output manually (no external library needed):
      - BEGIN:VCALENDAR, VERSION:2.0, PRODID:-//SIS QMS//Licenses//EN
      - For each event: BEGIN:VEVENT, UID, DTSTART (VALUE=DATE), SUMMARY, DESCRIPTION, END:VEVENT
      - END:VCALENDAR
    - Return Response with content_type='text/calendar' and Content-Disposition header
    - Use DTSTART;VALUE=DATE format (all-day events for expiration dates)

    Avoid: Installing icalendar library — the format is simple enough for string building.
    Avoid: Including past events — only future dates within the lookahead window.
  </action>
  <verify>Request GET /licenses/calendar.ics — returns valid .ics file. Import into Outlook or validate with online iCal validator. Verify events have correct dates matching license expirations.</verify>
  <done>AC-2 satisfied: iCal feed returns valid calendar with license expiration and CE deadline events</done>
</task>
